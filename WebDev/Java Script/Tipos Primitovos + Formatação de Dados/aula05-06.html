Tipos Primitivos e Tratamento de Dados

Basicamente, temos os tipos:

  • Number: para número (podendo ser Int (inteiro) ou Float(real))
  • Boolean: para true ou false
  • String: para cadeia de caracteres (Podem iniciar com ", ' ou `)

Alguns comandos da aula:

Para pop-up:

  • window.alert("Texto que aparece no pop-up") pop-up simples com OK
  • window.confirm('') pop-up com sim ou cancelar
  • window.prompt(``) pop-up com input

String -> Num

  • Number.parseInt("") transformar para número inteiro
  • Number.parseFloat("") transformar para número real
  • Number("") o JavaScript identifica qual o tipo

Num -> String

  • String(num)
  • num.toString()

Formatação de number

  • n.toFixed(2) para deixar com 2 casas decimais
  • n.toLocaleString('pr-BR', {style: 'currency', currency: 'USD'}) para tranformar o número em uma medida monetária de um determinado país!!
